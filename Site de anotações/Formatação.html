Formatação

Texto deletado

Podemos marcar um texto como excluído,usando a tag DEL (semantico), para indicar que ele deve ser lido, mas nao considerado

Texto inserido

Podemos marcar um texto como inserido, usando a tag INS (semantico), para dar ênfase e inidicar que ele foi adicionado depois

Existe tambem o sublinhado com a tag U (nao semantico)

Texto subrescrito

Para inserir coisas do tipo x20+3, usando a tag SUP

Texto subscrito

Para inserir coisas do tipo H2O, usando a tag SUB
